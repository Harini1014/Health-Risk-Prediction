#  Health Risk Prediction System

A Machine Learning-based web application that predicts patient health risk levels using vital parameters. The model is trained using a Random Forest Classifier and deployed using Flask for real-time predictions.


##  Features

- Predicts health risk levels:
  - 🟢 Normal
  - 🟢 Low
  - 🟡 Medium
  - 🔴 High
- User-friendly web interface
- Real-time prediction using Flask
- Handles missing values and categorical data
- High accuracy ML model


##  Machine Learning Details

- **Algorithm Used:** Random Forest Classifier
- **Training Accuracy:** 98.4%
- **Testing Accuracy:** 94.5%
- **Techniques Used:**
  - Data Cleaning
  - Missing Value Handling
  - Label Encoding
  - Feature Selection
  - Model Evaluation


##  Dataset Features

- Respiratory Rate
- Oxygen Saturation
- O2 Scale
- Systolic BP
- Heart Rate
- Temperature
- Consciousness
- On Oxygen
